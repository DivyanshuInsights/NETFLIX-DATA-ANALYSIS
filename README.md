### 👨‍💻 Created by Divyanshu Sharma

## 📌 Overview
This project presents an interactive **Netflix Data Analysis Dashboard** built using **Power BI**.  
It focuses on analyzing Netflix's content library (Movies & TV Shows) to uncover meaningful insights related to content distribution, trends, and ratings.

The objective of this project is to strengthen skills in **data analysis, visualization, and storytelling** by converting raw data into actionable insights.

---

## 📊 Key Insights

- 📁 **Total Content**
  - Movies: **7,814**
  - TV Shows: **3,031**

- 🎥 **Content Distribution**
  - Movies: **~72%**
  - TV Shows: **~28%**

- 🌍 **Top Countries**
  - United States contributes the highest content  
  - Followed by India, United Kingdom, Canada, etc.

- 📈 **Trend Analysis**
  - Significant growth in content after **2015**  
  - Peak additions observed around **2020–2021**

- 🔞 **Ratings Breakdown**
  - Most content falls under **TV-MA** and **TV-14**

---

## 🎛️ Dashboard Features

- Interactive filters (Slicers):
  - Type (Movie / TV Show)  
  - Country  
  - Release Year  
  - Rating  

- Visualizations included:
  - Donut Chart (Movies vs TV Shows)  
  - Bar Chart (Top Countries)  
  - Line/Area Chart (Titles by Year)  
  - Ratings Distribution Chart
